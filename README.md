# Student Performance Dataset Project 

This project analyzes a student performance dataset using Python and Google Colab.

## Features
- Read CSV dataset
- Display dataset information
- Analyze student performance
- Calculate averages and statistics
- Create charts and graphs
- Demonstrate sampling and distributions

## Technologies Used
- Python
- NumPy
- Seaborn
- Matplotlib
- SciPy
- Google Colab

## Dataset
The dataset contains information used for statistical analysis and visualization.

## Project Files
- `student_analysis.ipynb` → Main Colab notebook
- `student_performance.csv` → Dataset file

## Visualizations
The project includes:
- Distribution plots
- KDE plots
- Sample mean analysis
- Skewed distribution analysis

## Demonstration
The notebook demonstrates:
- Running Python code
- Loading the dataset
- Statistical calculations
- Creating charts and graphs
- Analyzing distributions
